## Project Title

CivicLens RAG - NYC 311 Operations Copilot

## Short Description

A local AI Data Engineering / Hybrid RAG project that extends an NYC 311 Lakehouse concept with cited document retrieval, PostgreSQL + pgvector search, predefined analytics summaries, and a Streamlit UI.

## Recruiter-Friendly Bullets

- Built an end-to-end local RAG pipeline with ingestion, chunking, deterministic embeddings, pgvector retrieval, cited answers, and safe no-answer behavior.
- Added a simple hybrid analytics path using predefined sample CSV outputs for questions like top complaint types, borough volume, agency volume, and backlog summary.
- Hardened the project with evaluation questions, pytest coverage, Dockerized pgvector, offline-safe CI, and clear local validation instructions.
